## рЯУМ Overview

The **Plagiarism Detection System** is a web-based application that analyzes submitted text or uploaded documents and compares them against a local corpus of reference documents.

It uses a **hybrid unsupervised machine learning approach** вАФ combining **TF-IDF Cosine Similarity** and **MinHash Jaccard Similarity** вАФ to produce a reliable ensemble plagiarism score with sentence-level match highlighting.

---

## вЬ® Features

| Feature | Description |
|:--|:--|
| рЯУД **Document Analysis** | Paste full text for document-level plagiarism scanning |
| рЯФН **Sentence Check** | Instantly check a single sentence against the entire corpus |
| рЯУБ **File Upload** | Upload `.txt`, `.pdf`, or `.docx` files for analysis |
| рЯЧВпЄП **Corpus Management** | Add, upload, or remove reference documents via the UI |

| рЯУК **Session History** | Track all checks performed during the current session |

---

## вЪЩпЄП How It Works

The detection engine applies two complementary similarity algorithms and combines them into a single ensemble score:

```
Ensemble Score = (0.6 √Ч Cosine Similarity) + (0.4 √Ч Jaccard Similarity)
```

| Algorithm | Weight | Description |
|:--|:--:|:--|
| **TF-IDF Cosine Similarity** | 60% | Measures vocabulary and term-frequency overlap |
| **MinHash Jaccard Similarity** | 40% | Approximates set-based shingle overlap via LSH |

### рЯЪ¶ Risk Level Thresholds

| Ensemble Score | Risk Level | Interpretation |
|:--:|:--:|:--|
| `< 0.26` | рЯЯҐ **Low** | Content appears mostly original |
| `0.26 вАУ 0.50` | рЯЯ° **Moderate** | Some sections may require review |
| `0.51 вАУ 0.75` | рЯЯ† **High** | Significant similarity detected |
| `вЙ• 0.76` | рЯФі **Critical** | Content is likely plagiarized |

## рЯМР API Reference

| Method | Endpoint | Description |
|:--:|:--|:--|
| `GET` | `/` | Serves the web interface |
| `POST` | `/check` | Analyze pasted text for plagiarism |
| `POST` | `/check_sentence` | Check a single sentence against the corpus |
| `POST` | `/upload` | Upload and analyze a file for plagiarism |

## рЯЪА Setup and Installation

### Prerequisites

- Python **3.9** or higher
- `pip` package manager

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/plagiarism-checker.git
cd plagiarism-checker
```

### 2. Create a Virtual Environment

```bash
# Windows
python -m venv venv
.\venv\Scripts\Activate.ps1

python -m venv venv
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open in Your Browser

```
http://127.0.0.1:5000
```

## рЯУЭ Notes

- Submission history is stored **in memory only** and resets when the server restarts
- The corpus loads `.txt` files from the `corpus/` directory at startup
- Uploaded PDF and DOCX files are extracted to plain text before being saved to the corpus
- For production use, consider replacing in-memory history with a database such as **SQLite** or **PostgreSQL**
